Documentation of Data used for experimental valiation of the EKF used in the paper titled 

"Online Estimation of 3-axis Magnetomenter Hard-Iron and Soft-Iron Bias Using Angular Velocity Sensors for Improved Dynamic Heading Accuracy: Theory and Experimental Evaluation"

Authors: Abhimanyu Shah, Andrew Spielvogel, Loius Whitcomb


The Folder includes the files

Microstrain Data:
2019_08_09_17-18.MST (Exp1)
2019_08_09_18-19.MST (Exp2)

read_microstrain.m (script to load microstain data on matlab)


Phins Data:
2019_08_09_17-18.INS (Exp1)
2019_08_09_18-19.INS (Exp2)

read_phins_imbat.m (script to load phins data on matlab)
